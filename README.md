# Xiao Zijian Portfolio Framework

Personal portfolio framework for Xiao Zijian, Visual Artist & Curator.

## Includes

- React + Vite
- Tailwind CSS
- React Router with `HashRouter` for GitHub Pages
- GSAP fan-style page transition
- CN / EN language switching
- Mock portfolio data for photography, easel paintings, exhibitions, and sketch works

## Local Preview

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Portfolio content can be updated in `src/data/portfolio.js`.
